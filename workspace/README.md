# 영감 인사이드 (Inspiration Inside)

디시 인사이드와 유사한 커뮤니티 웹페이지입니다. 순수 HTML, CSS, JavaScript로 구현되어 있으며, LocalStorage를 사용하여 오프라인에서도 동작합니다.

## 🌟 주요 특징

- **디시 인사이드와 유사한 UI/UX**: 친숙한 인터페이스
- **완전한 오프라인 동작**: 인터넷 연결 없이도 사용 가능
- **LocalStorage 기반**: 브라우저에 데이터 저장
- **반응형 디자인**: 모바일, 태블릿, PC 지원
- **완전한 커뮤니티 기능**: 회원가입, 로그인, 게시글, 댓글 등

## 📂 프로젝트 구조

```
영감인사이드/
├── index.html          # 메인 페이지
├── auth.html           # 로그인/회원가입 페이지
├── gallery.html        # 갤러리(게시판) 페이지
├── post.html           # 게시글 보기 페이지
├── write.html          # 글쓰기 페이지
├── mypage.html         # 마이페이지
├── css/
│   ├── style.css       # 공통 스타일
│   ├── main.css        # 메인 페이지 스타일
│   ├── auth.css        # 인증 페이지 스타일
│   ├── gallery.css     # 갤러리 페이지 스타일
│   ├── post.css        # 게시글 페이지 스타일
│   ├── write.css       # 글쓰기 페이지 스타일
│   └── mypage.css      # 마이페이지 스타일
├── js/
│   ├── common.js       # 공통 JavaScript
│   ├── main.js         # 메인 페이지 로직
│   ├── auth.js         # 인증 관련 로직
│   ├── gallery.js      # 갤러리 페이지 로직
│   ├── post.js         # 게시글 페이지 로직
│   ├── write.js        # 글쓰기 페이지 로직
│   └── mypage.js       # 마이페이지 로직

```

## 🚀 시작하기

### 1. 파일 다운로드
프로젝트 파일들을 로컬 컴퓨터에 다운로드합니다.

### 2. 브라우저에서 실행
`index.html` 파일을 더블클릭하거나 브라우저에서 열어주세요.

### 3. 첫 회원가입
- 우측 상단의 "회원가입" 버튼을 클릭
- 필요한 정보를 입력하여 계정 생성
- 생성된 계정으로 로그인

## 🎯 주요 기능

### 👤 사용자 관리
- **회원가입**: 아이디, 비밀번호, 닉네임으로 가입
- **로그인/로그아웃**: 세션 관리
- **프로필 수정**: 닉네임, 이메일, 자기소개 수정
- **비밀번호 변경**: 안전한 비밀번호 변경
- **계정 삭제**: 완전한 계정 삭제 기능

### 📝 게시판 시스템
- **다양한 갤러리**: 유머, 자유게시판, 뉴스, 기술, 게임
- **게시글 작성**: 마크다운 스타일 에디터
- **게시글 수정/삭제**: 작성자 권한 확인
- **임시저장**: 자동 임시저장 및 복구
- **미리보기**: 게시글 미리보기 기능

### 💬 댓글 시스템
- **댓글 작성**: 일반 댓글 및 익명 댓글
- **답글 기능**: 댓글에 대한 답글
- **댓글 추천/비추천**: 댓글 평가 시스템
- **댓글 삭제**: 작성자 댓글 삭제

### 🔍 검색 및 정렬
- **게시글 검색**: 제목, 내용, 작성자 검색
- **다양한 정렬**: 최신순, 인기순, 조회순, 댓글순
- **필터링**: 공지글, 일반글, 인기글 필터

### 👍 추천 시스템
- **게시글 추천/비추천**: 게시글 평가
- **댓글 추천/비추천**: 댓글 평가
- **중복 투표 방지**: 한 번만 투표 가능

### 📊 통계 및 관리
- **조회수 집계**: 중복 조회 방지
- **실시간 통계**: 갤러리별 게시글 수
- **마이페이지**: 내 글, 내 댓글 관리
