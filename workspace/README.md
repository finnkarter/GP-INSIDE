# GP-Inside - 오프라인 커뮤니티

인터넷 연결 없이도 완벽하게 작동하는 로컬 커뮤니티 사이트입니다.

## 🚀 주요 특징

- **완전 오프라인**: 인터넷 연결 없이도 모든 기능 이용 가능
- **로컬 저장**: 브라우저 로컬 스토리지를 활용한 데이터 영구 보존
- **갤러리 시스템**: 디시인사이드 스타일의 주제별 게시판 분리
- **회원 시스템**: 선택적 회원가입으로 추가 기능 이용 가능
- **권한 관리**: 회원/비회원별 차별화된 기능 제공
- **반응형 디자인**: 데스크톱과 모바일 모든 기기에서 최적화
- **모던 UI**: 깔끔하고 직관적인 사용자 인터페이스
- **최적화된 코드**: 최소한의 코드로 최대 효율 구현

## 🎨 기술 스택

- **HTML5**: 시멘틱 마크업
- **CSS3**: 모던 CSS (Grid, Flexbox, CSS Variables)
- **Vanilla JavaScript**: 프레임워크 없는 순수 자바스크립트
- **Local Storage**: 클라이언트 사이드 데이터 저장

│   ├── gallery.css     # 갤러리 관련 스타일
│   ├── post.css        # 게시글 및 댓글 스타일

## 🚀 사용 방법

1. **프로젝트 다운로드**
   - 모든 파일을 동일한 폴더에 저장

2. **실행**
   - `index.html` 파일을 웹 브라우저로 열기
   - 인터넷 연결 없이도 완벽 작동

3. **첫 사용**
   - 자동으로 샘플 갤러리와 게시글이 생성됩니다
   - 비회원으로도 게시글 작성 및 댓글 작성 가능

4. **회원가입 (선택사항)**
   - 우측 상단 "회원가입" 버튼 클릭
   - 일반 회원: 게시글 삭제, 닉네임 자동 입력
   
5. **관리자 로그인**
   - 아이디: `hamilton44`
   - 비밀번호: `lewishamilton44!`
   - 관리자 전용 기능: 갤러리 생성/삭제, 사용자 관리, 시스템 관리

## 💡 사용 팁

### 갤러리 이용
1. 메인 페이지에서 관심 있는 갤러리 선택
2. 갤러리별로 독립적인 게시판 운영
3. **관리자만** 새로운 갤러리 생성 가능

### 게시글 작성
1. 갤러리 선택 후 "글쓰기" 클릭
2. 제목, 작성자(회원은 자동), 내용 입력
3. "게시하기" 버튼으로 발행

### 검색 및 정렬
- 갤러리 검색: 메인 페이지에서 갤러리명으로 검색
- 게시글 검색: 각 갤러리에서 제목, 내용, 작성자 검색
- 드롭다운에서 정렬 방식 선택 가능

### 회원 전용 기능
- 🔸 마크로 회원 게시글/댓글 구분
- 본인 게시글 삭제 권한
- 닉네임 자동 입력

### 관리자 전용 기능
- 🔸 갤러리 생성/삭제 권한
- 🔸 모든 게시글/댓글 삭제 권한
- 🔸 사용자 관리 및 통계
- 🔸 시스템 로그 및 백업 기능
- 🔸 관리자 대시보드 접근

### 게시글 상호작용
- 게시글 클릭으로 상세 보기
- 좋아요 버튼으로 추천
- 댓글로 소통

### 다크/라이트 테마
- 🌙 우측 상단 토글 버튼으로 테마 전환
- ⌨️ 키보드 단축키: `Ctrl/Cmd + Shift + D`
- 🔄 시스템 테마 자동 감지 및 적용
- 💾 사용자 설정 로컬 저장

## 🔧 커스터마이징

### 색상 테마 변경
`css/base.css`의 `:root` 섹션에서 CSS 변수 수정:

```css
:root {
    --primary-color: #2563eb;      /* 메인 색상 */
    --background-color: #f8fafc;   /* 배경 색상 */
    --surface-color: #ffffff;      /* 카드 배경 */
    /* ... 기타 변수들 */
}
```

### 스타일 모듈별 수정
- **기본 스타일**: `css/base.css` - 변수, 리셋, 공통 컴포넌트
- **레이아웃**: `css/layout.css` - 헤더, 네비게이션, 그리드
- **갤러리**: `css/gallery.css` - 갤러리 카드, 폼 스타일
- **게시글**: `css/post.css` - 게시글 목록, 상세, 댓글
- **인증**: `css/auth.css` - 로그인, 회원가입 폼
- **관리자**: `css/admin.css` - 관리자 패널, 대시보드
- **테마**: `css/theme.css` - 다크/라이트 테마, 전환 효과
- **반응형**: `css/responsive.css` - 모바일, 태블릿 최적화

### 기능 확장
- **JavaScript 모듈**: `js/` 폴더의 각 모듈 확장
- **새 모듈 추가**: 새로운 `.js` 및 `.css` 파일 생성 후 `index.html`에 추가
